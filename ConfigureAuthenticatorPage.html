Configure authenticator

To use an authenticator app go through the following steps:
  1. Download a two-factor authenticator app like Microsoft Authenticator
    for Android and iOS or Google Authenticator for Android and iOS.
  2. Scan the QR code using the authenticator app.
  3. Once you have scanned the QR code or input the key above, your two factor
    authentication app will provide you with a unique code. Enter the code in the
    confirmation box below.
